# 한글 IME 수동 재현 체크리스트

Stcode 입력창(`ChatInput`)은 macOS IME가 주는 UTF-16 range를 UTF-8 byte offset으로
변환해서 처리한다. 이 체크리스트의 목적은 한글 조합 중 char boundary panic, 잘못된
선택 범위, 의도치 않은 전송을 빠르게 잡는 것이다.

## 준비

- macOS 입력 소스: `한국어 - 두벌식`.
- 앱 실행: `bash scripts/build-app.sh --debug --no-codesign` 후 `dist/Stcode.app` 실행.
- 테스트 대상: 채팅 입력창. 설정 모달의 Provider/조율 모델/작업 모델 입력도 같은 `ChatInput`이므로
  마지막에 한 번만 확인한다.
- 실패 기록에는 입력 문자열, 조작 순서, 기대/실제 결과, 앱 로그를 함께 남긴다.

## 기본 조합

- [ ] `안녕하세요`를 천천히 입력한다.
  - 기대: 조합 중 글자가 깨지지 않고, 완료 후 정확히 `안녕하세요`가 남는다.
- [ ] `한글abc123`을 입력한다.
  - 기대: 한글과 ASCII 사이에서 커서 위치가 밀리지 않는다.
- [ ] `한🙂글`을 입력하거나 붙여넣는다.
  - 기대: emoji 앞뒤 이동/삭제 시 panic 없이 한 글자 단위로 처리된다.

## 조합 중 편집

- [ ] `한그`까지 조합하다가 Backspace를 누른 뒤 `글`로 고쳐 입력한다.
  - 기대: 마지막 조합 글자만 자연스럽게 수정되고 앞 글자는 보존된다.
- [ ] 조합 중 Left/Right를 눌러본다.
  - 기대: 앱이 멈추지 않고, 커서가 char boundary 밖으로 나가지 않는다.
- [ ] 조합 중 Esc 또는 입력 소스 전환으로 조합을 취소한다.
  - 기대: 취소/확정 상태가 macOS IME 동작과 맞고, 다음 입력이 기존 글자를 덮어쓰지 않는다.

## 선택과 치환

- [ ] `가나다라마바사` 입력 후 Shift-Left를 3번 누른다.
  - 기대: 마지막 세 글자만 선택된다.
- [ ] 선택된 상태에서 `테스트`를 입력한다.
  - 기대: 선택 영역만 `테스트`로 치환된다.
- [ ] 마우스로 `나다라` 범위를 드래그 선택한다.
  - 기대: 드래그 방향과 관계없이 선택 범위가 정확하다.
- [ ] `Cmd-A` 후 `새 프롬프트`를 입력한다.
  - 기대: 전체 내용이 새 문장으로 교체된다.

## 클립보드

- [ ] `한글 복사 테스트` 일부를 선택하고 `Cmd-C` 후 외부 편집기에 붙여넣는다.
  - 기대: 선택한 문자열만 복사된다.
- [ ] 선택 영역에 `Cmd-V`로 `붙여넣기 한글`을 넣는다.
  - 기대: 선택 영역만 교체되고 커서는 붙여넣은 텍스트 뒤로 이동한다.
- [ ] 여러 줄 텍스트를 붙여넣는다.
  - 기대: 현재 입력창은 single-line 정책이므로 줄바꿈이 공백으로 바뀐다.
- [ ] `Cmd-X`로 한글 선택 영역을 잘라낸다.
  - 기대: 선택 영역이 삭제되고, 잘라낸 내용이 클립보드에 들어간다.

## 전송 경계

- [ ] 한글 조합이 끝난 상태에서 Enter를 누른다.
  - 기대: prompt가 전송되고 입력창이 비워진다.
- [ ] 한글 조합 중 Enter를 누른다.
  - 기대: 우선 조합 확정이 되어야 한다. 조합 중간 문자열이 바로 전송되면 버그로 기록한다.
- [ ] 전송 직후 새 한글 prompt를 바로 입력한다.
  - 기대: 이전 prompt의 marked range가 남아 새 입력을 덮어쓰지 않는다.

## 설정 모달

- [ ] 설정 모달을 열고 Provider/조율 모델/작업 모델 입력칸에서 값을 지운 뒤 다시 입력한다.
  - 기대: ASCII 편집이 정상이다.
- [ ] 같은 입력칸에 임시로 `테스트`를 입력한 뒤 전체 선택/삭제한다.
  - 기대: 한글 selection/delete가 채팅 입력창과 동일하게 동작한다.

## 통과 기준

- 앱 panic/crash 없음.
- 한글 조합 중 문자열이 깨지거나 중복 삽입되지 않음.
- 커서 이동, Backspace/Delete, selection, clipboard가 한글 char boundary를 침범하지 않음.
- 조합 중 Enter가 의도치 않게 prompt를 전송하지 않음.
